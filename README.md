# Vagrant - Kali

Vagrantfile configuration to run a Kali Linux system on VirtualBox using Vagrant.

## Requirements

- [Vagrant](https://www.vagrantup.com/docs/cli/)

## Deployment

```
vagrant up --provision
```

## Accessing the machine

```
vagrant ssh
```
